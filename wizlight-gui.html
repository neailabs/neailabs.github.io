WizLight GUI Controller

A modern, intuitive desktop application for controlling Wiz smart lights. Built with PySide6 and Python, featuring automatic device discovery, RGB color control, scene management, and real-time status updates.

Download View Source

Key Features

Auto Discovery

Automatically detect Wiz lights on your network with multi-network scanning and intelligent broadcast IP detection.

RGB Color Control

Full RGB color control with intuitive sliders, color picker dialog, and real-time color preview for perfect lighting.

32 Built-in Scenes

Choose from 32 pre-configured lighting scenes including Ocean, Sunset, Party, Forest, and many more atmospheric options.

Color Temperature

Adjust white light temperature from warm 1000K to cool 8000K for the perfect ambiance at any time of day.

Animation Speed Control

Fine-tune the speed of dynamic scenes and color-changing effects with precision speed control from 0-100%.

Real-time Updates

Automatic status monitoring with 5-second refresh intervals to keep your interface synchronized with your lights.

Dynamic Scene Preview

Experience smooth color transitions and dynamic lighting effects with our advanced scene engine.

Quick Setup

Prerequisites

# Python 3.8+ and PySide6 required
pip install PySide6>=6.4.0

Installation

# Clone the repository
git clone https://github.com/yourusername/wizlight-gui.git
cd wizlight-gui

# Install dependencies
pip install -r requirements.txt

# Build the C++ backend (or download for macOS M-series)
mkdir build && cd build
cmake .. && make

# Run the application
cd wizlight-gui
python main.py

macOS Users

# For M-series Macs, download pre-built binary
# Download from: https://www.neailabs.com/download/1006/?tmstv=1755590204
# Place in ../build/ directory

# Create macOS app bundle
python build_macos.py
